River
=====

River is a set of low-level libraries for building *computational graphs*.

It is designed to be *massively distributed*, *fault-tolerant*, extendable, and 
easy to use.

Getting started
---------------

* Install: `npm i`
* Build (all modules): `grunt build`
* Test (all modules): `grunt test`
